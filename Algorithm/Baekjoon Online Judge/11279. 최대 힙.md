# BOJ(우선순위 큐) - 11279. 최대 힙

## 1. 문제  
https://www.acmicpc.net/problem/11279
## 2. 접근 방법  
우선순위 큐를 이용해 최대 힙 구현
## 3. 코드  
```
#include <iostream>
#include <queue>
using namespace std;

int main()
{
	cin.tie(NULL);
	ios_base::sync_with_stdio(false);

	int N;
	cin >> N;

	priority_queue<int, vector<int>, less<int> > pq;
	for (int i = 0; i < N; i++)
	{
		int val;
		cin >> val;

		if (val == 0)
		{
			if (pq.empty())	cout << "0\n";
			else
			{
				cout << pq.top() << "\n";
				pq.pop();
			}
		}
		else
			pq.push(val);
	}

	return 0;
}
```
